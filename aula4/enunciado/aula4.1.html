Exercício 1 — DeadlineService

Criar um sistema de prazos (deadlines) para tarefas.

Dicas:
- Criar ficheiro services/deadlineService.ts
- Associar uma data limite a cada task
- Não alterar a classe Task existente

Funções esperadas:

- isExpired(taskId)
- getExpiredTasks()

Exercício 10 — AutomationRulesService

Sistema de regras automáticas do projeto.

Dicas:
- Não alterar classes existentes
- Apenas aplicar regras por serviços
- Criar ficheiro services/automationRulesService.ts

Regras:
- Se task ficar COMPLETED → criar log automático
- Se task ficar BLOCKED → notificar
- Se user ficar inactive → remover assignments
- Se task expirar → mover para BLOCKED

Funções:
- applyRules(task)
- applyUserRules(user)
